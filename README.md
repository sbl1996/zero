# 零 ZERO

一个基于 Vue 3 的网页单机玄幻 RPG 游戏。

## 快速开始

### 获取素材
请前往 [素材仓库](xxx) 下载游戏素材，并将其解压到 `src/assets` 目录下。

### 安装运行
```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build
```

## 游戏特性

- 🎮 **角色养成** - 创建角色，修炼提升境界
- 🗺️ **地图探索** - 探索不同区域，挑战怪物
- ⚔️ **战斗系统** - 回合制战斗，使用技能与装备
- 📿 **修炼系统** - 冥想恢复，使用晶核加速
- 🎒 **物品装备** - 收集装备、道具和材料
- 📋 **任务系统** - 完成任务获得奖励

## 技术栈

- Vue 3 + TypeScript
- Pinia (状态管理)
- Vue Router
- Vite (构建工具)
