# 零 ZERO

一个基于 Vue 3 的网页单机玄幻 RPG 游戏。

## 快速开始

### 获取素材
请前往 [素材仓库](xxx) 下载游戏素材，并将其解压到 `src/assets` 目录下。

### 安装运行
```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build
```

## 游戏特性
- AI NPC：由大模型驱动的 NPC，支持语音对话、任务、交易等。
- 音游化魔法：通过类音游的方式演奏乐曲，释放魔法。
- 修炼系统：无等级，更接近于玄幻小说世界的修炼方式。

## 技术栈

- Vue 3 + TypeScript
- Pinia (状态管理)
- Vue Router
- Vite (构建工具)
- Tone.js (音游化魔法)
